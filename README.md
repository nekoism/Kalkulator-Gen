Python day 2 : Kalkulator Gen
Program sederhana menggunakan Python untuk mengecek generasi seseorang
berdasarkan tahun kelahiran.

## Fitur
- Input tahun kelahiran
- Menampilkan generasi:
  - Silent Generation
  - Baby Boomers
  - Gen X
  - Milenial
  - Gen Z
  - Gen Alpha

## Cara Menjalankan
1. Pastikan Python sudah terinstall
2. Jalankan file dengan perintah:
   python main.py
3. Masukkan tahun kelahiran saat diminta

## Contoh Output
Masukkan tahun kelahiran: 1999  
Gen Z
